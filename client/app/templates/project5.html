GFS Model Output Portfolio Project

I designed and implemented a cloud-based (AWS) workflow for generating GFS model images using Apache Airflow to orchestrate ECS tasks. The image generation code, written in Python, is packaged in an ECR container and deployed via AWS Fargate. I leveraged Terraform to automate infrastructure provisioning, including building and pushing the container image, configuring the Fargate cluster, defining the task and service, and deploying an Airflow instance on EC2. This solution enhances automation, scalability, and operational efficiency in weather model visualization.

You can view the repository here.

The images are updated four times a day.
